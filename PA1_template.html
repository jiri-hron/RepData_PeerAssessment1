Loading and preprocessing the data

The unzip function is used to extract the data file to a temporary directory created by tempdir function (the directory will be automatically deleted by the anonymous function on.exit). Once the activity.csv file is extracted, it is loaded by read.csv function. The code assumes the archive with the original data to be placed in the current working directory.

# create the temporary directory
tmpDir = tempdir();
# ensure it's deleted when the script is finished
on.exit(function() unlink(tmpDir, TRUE, TRUE));
# ensure it really was created (sometimes help on Debian-based distros)
dir.create(tmpDir, recursive=TRUE, showWarnings=FALSE);

# unzip the file to the temporary directory
unzippedDir = file.path(tmpDir, "unzipped");
dir.create(unzippedDir, showWarnings=FALSE);
unzip(zipfile="./activity.zip",exdir=unzippedDir);

# read the file
filePath = file.path(unzippedDir, "activity.csv");
movements = read.csv(filePath,
                     header = TRUE,
                     sep = ",",
                     quote = "\"",
                     comment.char = "",
                     stringsAsFactors = FALSE,
                     colClasses = c("integer", "Date", "integer"));

What is mean total number of steps taken per day?

The histogram shows the total number of steps taken per day. The total number of steps per day is summed up, ignoring all the missing values. The Freedman- Diaconis rule is used for computing the breaks.

stepsPerDay = sapply(split(movements$steps, movements$date),
                     sum,
                     na.rm = TRUE);
hist(stepsPerDay,
     main = "Total steps per day",
     xlab = "Number of steps",
     ylab = "Number of days",
     breaks = "fd",
     ylim = c(0, 30),
     col = "#FF5575");

The mean and median values of total numner of steps per day are also calculated disregard of all the NA values.

Imputing missing values

The total number of rows with at least one missing value:

# use the underlying representation of logical vector, 0 for FALSE, 1 for TRUE
nrow(movements) - sum(complete.cases(movements));

## [1] 2304

Fill the missing values with the average number of steps taken during the period at other days.

fillAvgInterVal = function(mat) {
  stepCol = avgStepsInterval[mat[,"interval"]];

  for(i in which(!complete.cases(movements))) {
    mat[i,]["steps"] = stepCol[i];
    }
  mat
  }

Use the code above to create a new data set with the missing values filled in.

filledMov = fillAvgInterVal(movements);

Once again calculate the total sum of steps per day with the missing values filled in.

fstepsPerDay = sapply(split(filledMov$steps, filledMov$date),
